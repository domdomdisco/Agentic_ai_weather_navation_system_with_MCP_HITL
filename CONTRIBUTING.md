# Contributing to The Agentic Bridge

First off, thank you for considering contributing to The Agentic Bridge! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tab to report bugs.
- Provide a clear and descriptive title.
- Describe the exact steps which reproduce the problem.

### Suggesting Enhancements
- Check the issues to see if the enhancement has already been suggested.
- Explain why this enhancement would be useful to most users.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code follows the existing style.

## Style Guide
- We follow [PEP 8](https://www.python.org/dev/peps/pep-0008/) for Python code.
- Write clear, concise commit messages.
